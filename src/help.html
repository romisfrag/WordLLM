WordLLM Help Guide

Getting Started

Installation

  1. Open Word
  2. Go to Insert > Add-ins > My Add-ins
  3. Choose "Upload My Add-in"
  4. Browse to the WordLLM manifest file
  5. Click "Open"

Features

AI-Powered Writing Assistance

WordLLM provides various AI-powered features to enhance your writing:

  • Text generation and completion
  • Grammar and spelling correction
  • Style improvement suggestions
  • Content summarization

Configuration

Setting Up Your API Keys

  1. Open the WordLLM task pane
  2. Click on the settings icon
  3. Enter your API key for your preferred LLM provider
  4. Save your settings
Note: You'll need to obtain API keys from your preferred LLM provider (e.g., OpenAI, openrouter, etc.) to use WordLLM's features.

Troubleshooting

Common Issues

  • Add-in not loading: Ensure you have a stable internet connection
  • API errors: Verify your API key is correct and has sufficient credits
  • Slow response: Check your internet connection and try again
